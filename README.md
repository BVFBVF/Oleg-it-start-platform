# Oleg-it-start-platform

Стек: Angular, TypeScript, RxJS, NgRx Signals, SCSS

Моковые данные придумай сам в src/app/mock-data.ts

## Фича 1: Дашборд сотрудников /dashboard

- Список сотрудников в виде карточек
- На карточке: имя, аватар, статус
  (active / blocked / finished), стек (теги),
  readiness score (прогресс-бар)
- Фильтрация по статусу через RxJS
- Поиск по имени (debounceTime 300ms)
- Клик на карточку → страница сотрудника
- Стейт через NgRx Signals

## Фича 2: Страница сотрудника /employee/:id

- Имя, аватар, стек (теги), контакты
- История задач: название, статус, дедлайн
- Список PR: название, ссылка,
  статус (merged / open / closed)
- Замечания ментора: список комментов с датой
- Кнопка "Назад к списку"

## Требования
- Standalone компоненты (без NgModules)
- OnPush стратегия везде
- Lazy loading для роутов
- RxJS для асинхронных операций
- NgRx Signals для глобального стейта
- Юнит-тесты на один компонент (Vitest)
- SCSS, адаптивная вёрстка
- Каждая фича — отдельная ветка и PR в main
